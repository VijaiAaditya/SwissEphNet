# SwissEphNet

This project is an Astrodienst Swiss Ephemeris (http://www.astro.com/swisseph/) .Net portage from 
C (version 2.06) to C# in a PCL/.Net Core project for cross platform usage.

WASM .NET CORE 9 upgradation
